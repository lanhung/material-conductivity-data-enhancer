# ZrO₂ 电导率数据生成方案（1亿条，遵循物理规律）

## Context

当前项目拥有 1,351 条真实锆基材料电导率数据，分布在 4 张关联表中。目标是生成 1 亿条符合物理规律的合成数据用于机器学习训练。表结构已确定（material_samples、sample_dopants、sintering_steps、sample_crystal_phases）。本方案专注于**如何保证生成数据的物理合理性**。

---

## 一、生成架构：以"配方组"为基本单位

**核心设计**：生成单位不是单条样本，而是**一个配方组（Recipe Group）**。

一个配方组 = 一种材料组成（掺杂 + 工艺 + 晶相），对应多个测试温度点。

- 1亿条样本 ≈ **~2000-3000 万个配方 × 平均 3-5 个温度点**
- 同一配方组内的样本共享相同的掺杂信息、烧结条件、晶相，只有 operating_temperature 和 conductivity 不同
- 电导率严格遵循 Arrhenius 关系在组内单调递增

---

## 二、生成顺序与各步骤物理约束

### 步骤 1：生成掺杂信息（sample_dopants）

**1.1 掺杂元素数量**
- 分布：1(46%), 2(46%), 3(5%), 4(2%), 5(1%)

**1.2 元素选择**
- 从 20 种元素中按真实频率加权采样：Y(38%), Sc(33%), Yb(9%), Ce(5%), Dy(3%), Bi(3%), ...
- 同一配方内元素不重复
- **硬约束（规律4）**：至少含一种 +2 或 +3 价元素。若随机到纯 +4/+5 价组合，必须重新采样

**1.3 离子半径与价态**
- **硬约束（规律2）**：由查找表 DOPANT_PROPERTIES 固定映射，不可随机

**1.4 摩尔分数**
- 主掺杂：从 Beta 分布采样，中心在各元素的最优浓度附近
  - Y: Beta(α=4, β=3) × 0.20 → 中心约 0.08
  - Sc: Beta(α=5, β=3) × 0.15 → 中心约 0.09
  - Ca: Beta(α=3, β=2) × 0.25 → 中心约 0.12
- 次掺杂（共掺杂）：从 Beta(2,5) × 0.08 采样 → 中心约 0.02
- **硬约束（规律6）**：SUM(molar_fraction) ≤ 0.30，否则裁剪
- **硬约束（规律5）**：各元素不超过其固溶度上限
  - Sc ≤ 0.12, Ce ≤ 0.18, Y ≤ 0.25（实际数据很少超过）, Ca ≤ 0.20
- **硬约束**：每条 molar_fraction > 0

### 步骤 2：生成晶相信息（sample_crystal_phases）

**依赖步骤 1 的掺杂信息**

**2.1 晶相数量**
- 分布：1(84%), 2(16%), 3(<1%)

**2.2 主相选择 — 条件概率依赖掺杂浓度（规律3）**

计算 total_dopant_fraction 和 weighted_avg_valence：

| 条件 | 主相概率分布 |
|------|------------|
| 总掺杂 ≥ 0.10 且主掺杂为 +3 价 | c:80%, t:15%, m:5% |
| 总掺杂 0.05-0.10 | c:40%, t:45%, m:10%, o:5% |
| 总掺杂 < 0.05 | c:5%, t:40%, m:45%, o:10% |
| 主掺杂为 Sc 且 Sc ≥ 0.10 | c:85%, t:10%, r:5% |
| 主掺杂为 Sc 且 Sc < 0.08 | c:10%, t:50%, r:25%, m:15% |

**硬约束**：
- 低掺杂(总<0.05) + 纯立方 → 禁止
- 高掺杂(总>0.12) + 纯单斜 → 禁止
- 每个样本恰好 1 个 is_major_phase=1

**2.3 次相（如果有）**
- 单相样本：无次相
- 双相样本：次相从与主相相邻的相中选择
  - 主相 c → 次相 t
  - 主相 t → 次相 c(60%) 或 m(40%)
  - 主相 m → 次相 t

### 步骤 3：生成烧结信息（sintering_steps）

**依赖主表的 synthesis_method 和 processing_route**

**3.1 烧结步数**
- 分布：1(97%), 2(2%), 3(1%)；3% 无烧结数据

**3.2 烧结温度与时长 — 依赖合成方法（规律9）**

| synthesis_method + processing_route | 温度范围 (°C) | 时长范围 (min) |
|-------------------------------------|-------------|---------------|
| 固相合成 + 干压 | 1400-1650 | 120-600 |
| 商业粉 + 干压 | 1300-1550 | 60-300 |
| 溶胶凝胶 + 干压 | 1200-1500 | 60-360 |
| 共沉淀 + 干压 | 1200-1500 | 60-360 |
| * + SPS | 1000-1300 | 3-30 |
| * + RF溅射 | 无烧结 | 无 |

**温度采样**：70% 从真实分布采样，30% 在对应范围内均匀采样
**时长采样**：类似混合分布

### 步骤 4：生成主表信息（material_samples）— 除电导率外

**4.1 固定字段**
- sample_id: 从 10,000,001 递增
- reference: 'SYNTHETIC'

**4.2 合成方法与工艺**
- synthesis_method: 按真实分布采样（固相合成 45.6%, 商品化 20%, 溶胶凝胶 12%, 共沉淀 10%, 水热 5%, 其他 7.4%）
- processing_route: 按真实分布采样（干压 80.4%, SPS 8%, RF溅射 3%, 其他 8.6%）

**4.3 工作温度序列（同一配方组内多个温度点）**
- 每个配方组生成 3-5 个温度点（分布：3个:40%, 4个:30%, 5个:20%, 6-8个:10%）
- 温度范围：[300, 1400]°C
- 采样策略：
  - 每组先确定一个中心温度（从真实分布采样）
  - 然后在中心温度 ± 200°C 范围内等间距或随机生成其他温度点
  - 间距 50-100°C

### 步骤 5：生成电导率（conductivity）— 核心物理约束

**这是最关键的步骤，必须满足多条物理规律。**

**方案：分段 Arrhenius + LightGBM 混合**

**5.1 为每个配方组确定 Arrhenius 参数**

对于每个配方组，需要确定两组参数（分段 Arrhenius，规律1修正）：
- 低温段（<600°C）：Eₐ_low, σ₀_low
- 高温段（≥600°C）：Eₐ_high, σ₀_high

**Eₐ 的确定依据（规律8、13）：**

| 主掺杂元素 | Eₐ_high (eV) | Eₐ_low (eV) |
|-----------|-------------|-------------|
| Sc | 0.78-0.85 | 1.05-1.15 |
| Y | 0.90-1.05 | 1.10-1.25 |
| Yb | 0.85-0.95 | 1.05-1.20 |
| Gd | 0.95-1.10 | 1.10-1.30 |
| Dy | 0.95-1.10 | 1.10-1.30 |
| Ca | 1.00-1.20 | 1.15-1.35 |
| 其他 | 0.95-1.15 | 1.10-1.30 |

- Eₐ 在对应范围内随机采样（正态分布，中心为范围中点）
- 掺杂浓度偏离最优值越远 → Eₐ 越高（规律7）

**σ₀ 的确定**：
- 用 LightGBM 模型在一个参考温度（如 800°C）预测 log10(conductivity)
- 再用 Arrhenius 公式反推 σ₀
- 这样既利用了 ML 模型学到的组成-电导率关系，又保证了温度序列的物理一致性

**5.2 生成电导率序列**

对配方组内的每个温度 T_i：
1. 根据 T_i 选择对应的 Eₐ 和 σ₀（高温段或低温段）
2. 计算 σ(T_i) = σ₀/T_i × exp(-Eₐ/kᵦT_i)
3. 加微小噪声：log10(σ) += N(0, 0.05)  （σ=0.05 的高斯噪声在 log 空间）
4. 裁剪到 [1e-8, 1.0] S/cm

**5.3 物理趋势修正（软约束检查）**

生成后对每个配方组检查：
- **规律1**：温度序列中电导率严格单调递增 → 如因噪声违反，重新排序噪声
- **规律8/电导率排序**：Sc 体系整体电导率 > Y 体系 > Gd/Dy 体系 → 不需要逐条检查，但可抽样验证统计趋势
- **规律10/晶界效应**：烧结温度 < 1300°C 的配方，电导率乘以衰减因子 0.5-0.8
- **规律11/ScSZ退化**：Sc为主掺杂 且 Sc<10% 且 T<650°C → 电导率额外降低 50-80%
- **规律12/共掺杂非加性**：掺杂元素 ≥ 3 → 电导率乘以衰减因子 0.7-0.9
- **规律13/过渡金属**：含 Fe/Mn → 电导率乘以增强因子 1.1-1.3

### 步骤 6：生成 material_source_and_purity（模板化）

沿用 plan2 的模板方案，按 synthesis_method 选择模式：
- 商业粉体模式 (70%)
- 自制粉体模式 (15%)
- 简略描述模式 (15%)

## 三、14 条物理规律在生成流程中的落实位置

| # | 物理规律 | 落实步骤 | 实现方式 |
|---|---------|---------|---------|
| 1 | Arrhenius 温度依赖 | 步骤5 | 分段 Arrhenius 公式生成温度序列 |
| 2 | 离子半径/价态固定 | 步骤1 | DOPANT_PROPERTIES 查找表 |
| 3 | 晶相-掺杂耦合 | 步骤2 | 条件概率表 + 硬约束过滤 |
| 4 | 价态-氧空位 | 步骤1 | 采样后检查，纯+4/+5重采样 |
| 5 | 固溶度上限 | 步骤1 | 各元素上限裁剪 |
| 6 | 总掺杂上限 | 步骤1 | SUM ≤ 0.30 裁剪 |
| 7 | 浓度非单调性 | 步骤5 | Eₐ 随浓度偏离最优值增加 |
| 8 | 离子半径-电导率 | 步骤5 | Eₐ 按元素类型分配 |
| 9 | 烧结-合成方法耦合 | 步骤3 | 条件分布表 |
| 10 | 晶界效应 | 步骤5 | 低烧结温度衰减因子 |
| 11 | ScSZ 相退化 | 步骤5 | Sc<10%低温额外衰减 |
| 12 | 共掺杂非加性 | 步骤5 | ≥3种掺杂衰减因子 |
| 13 | 过渡金属电子电导 | 步骤5 | Fe/Mn 增强因子 |
| 14 | 薄膜特殊性 | 步骤3/4 | RF溅射无烧结，温度范围调整 |

## 四、数据量规划

| 表 | 每配方组平均行数 | 每样本平均行数 | 预计总行数 |
|---|---------------|-------------|----------|
| material_samples | 3-5 (温度点) | 1 | 1 亿 |
| sample_dopants | 1.68 × (3-5) | ~1.68 | ~1.68 亿 |
| sintering_steps | 1 × (3-5) | ~1 | ~1 亿 |

配方组数 ≈ 1亿 / 平均温度点数(~4) ≈ **2500万个配方组**

## 五、验证清单

### 硬约束验证（必须 100% 通过）
1. 所有 conductivity > 0
2. 所有 molar_fraction > 0 且 ≤ 对应元素固溶度上限
3. 每个样本的 SUM(molar_fraction) ≤ 0.30
4. 每个样本至少含一种 +2 或 +3 价掺杂
5. 每个样本恰好 1 个 is_major_phase = 1
6. 子表 sample_id 全部存在于 material_samples
7. 同一配方组内，温度↑ → 电导率严格↑
8. 低掺杂(<0.05)不出现纯立方；高掺杂(>0.12)不出现纯单斜

### 统计分布验证（抽样检查）
1. 掺杂元素频率分布与真实数据相似
2. operating_temperature 分布与真实数据相似
3. conductivity 对数分布与真实数据相似
4. corr(operating_temperature, ln(conductivity)) ≈ 0.7-0.8
5. Sc 体系平均电导率 > Y 体系 > Gd 体系（同温度同浓度下）
